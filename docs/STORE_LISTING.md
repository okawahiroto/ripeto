```
練習するたびに星が育つ、逆算型の楽器練習記録アプリ。発表会・演奏会・コンクールまであと何日か、いつでも確認できます。
```

### 説明文（4,000文字以内）
```
【本番まで、逆算して練習しよう】

発表会・演奏会・コンクール――「あと何日で本番？」を常に意識しながら練習できるアプリです。

■ こんな方に
・発表会や演奏会に向けて練習している方
・どこをどれだけ練習したか記録を残したい方
・練習の積み上げを達成感として味わいたい方

■ 主な機能

【カウントダウン】
本番の日付を登録すると、今日から何日後かがひと目でわかります。複数の目標を同時に管理することも可能です。

【曲目・練習箇所の管理】
曲ごとに「冒頭8小節」「サビ」など練習箇所を自由に登録。1タップで練習回数を記録できます。

【星の成長】
練習回数に応じて星が進化します。
・1〜9回：黄色の星
・10〜99回：ゴールドの星
・100回達成：虹色キラキラ星＋お祝いアニメーション！

【オフライン対応】
練習室や地下のスタジオでも快適に使えます。電波がなくても記録が残ります。

■ プライバシーに配慮した設計
アカウント登録不要で今すぐ使えます。氏名・メールアドレスなどの個人情報の入力は一切不要です。Google / Apple アカウントとの連携は任意で、機種変更時のデータ引き継ぎに使えます。

■ プレミアム（買い切り）
一度の購入で広告を完全非表示にできます。サブスクリプション不要。機種変更後も購入の復元が可能です。

アプリ名の「Ripeto（リペート）」は音楽記号で「繰り返し」を意味します。日々の練習を積み重ねて、最高の本番を迎えましょう。
```

### キーワード（100文字以内、カンマ区切り）

## スクリーンショット撮影ガイド

### App Store 必須サイズ
| デバイス | 解像度 | 必須 |
|---|---|---|
| iPhone 6.9"（iPhone 16 Plus） | 1320 × 2868 px | ✅ 必須 |
| iPhone 6.7"（iPhone 14 Plus/15 Plus） | 1290 × 2796 px | 推奨 |
| iPad Pro 12.9" | 2048 × 2732 px | タブレット対応時 |

### Google Play 必須サイズ
- スマートフォン: 最低2枚、16:9 or 9:16

### 撮影する画面（推奨順）
1. **ダッシュボード**（カウントダウンが見える状態）
2. **ゴール一覧**（複数のゴールが並んでいる状態）
3. **曲目・練習箇所一覧**（星が育っている状態）
4. **練習ログ入力**（入力の手軽さを訴求）
5. **星の演出**（100回達成の虹色キラキラ）

### 撮影手順（iOS Simulator）
```bash
# 正しいシミュレータを起動
npx expo run:ios --device "iPhone 16 Plus"

# スクリーンショットはシミュレータの Cmd + S で保存
```

## プライバシーポリシーのホスティング手順（GitHub Pages）

1. GitHub に `ripeto` リポジトリ（または専用の `okawahiroto.github.io` リポジトリ）を作成
2. `/docs/privacy.html` に `app/legal/privacy.tsx` の内容を HTML 変換して配置
3. GitHub Pages を有効化（Settings → Pages → Source: main / docs）
4. URL `https://okawahiroto.github.io/ripeto/privacy` を App Store Connect に登録

> 代替案: [Privacy Policy Generator](https://www.privacypolicygenerator.info/) などの無料サービスで hosted URL を取得する方法もある。
