# Roadmap

## v1.0 — リリース済み（2026-05-07 iOS App Store 公開）

### Phase 1: プロジェクト基盤
- [x] Expo プロジェクト初期化（TypeScript + Expo Router）
- [x] NativeWind セットアップ
- [x] Firebase プロジェクト作成・接続
- [x] 環境変数管理（`.env` + `app.config.ts`）
- [x] ESLint / Prettier 設定

### Phase 2: 認証
- [x] Firebase 匿名認証
- [x] 認証状態管理（Zustand）
- [ ] Google ログイン（任意リンク）※ 審査対応のため v1.0 では削除、v1.1 で再実装予定
- [ ] Apple ログイン（任意リンク）※ 同上

### Phase 3: コア機能（CRUD）
- [x] ゴール作成・編集・一覧・完了マーク
- [x] 曲目 CRUD
- [x] 練習箇所 CRUD
- [x] 練習ログ追加（テキスト + 星カウントアップ）

### Phase 4: ダッシュボード
- [x] 残り日数カウントダウン（直近3件）
- [x] 各種カウンター表示（ゴール数・曲数・練習箇所数）
- [x] 練習回数の星表示

### Phase 5: 星演出
- [x] 段階別の星表示コンポーネント
- [x] 100回到達時のLottieアニメーション
- [x] 10回到達時の控えめな演出

### Phase 6: 収益化
- [x] AdMob 統合（バナー広告）
- [x] RevenueCat 統合
- [x] 買い切り課金（広告非表示）
- [x] 購入復元機能

### Phase 7: リリース準備
- [x] アプリアイコン・スプラッシュ画面
- [x] プライバシーポリシー
- [x] 利用規約
- [x] App Store / Google Play 申請準備
- [x] EAS Build 設定
