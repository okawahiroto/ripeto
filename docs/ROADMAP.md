# Roadmap

## MVP (v1.0) - 最初のリリース

### Phase 1: プロジェクト基盤
- [ ] Expo プロジェクト初期化（TypeScript + Expo Router）
- [ ] NativeWind セットアップ
- [ ] Firebase プロジェクト作成・接続
- [ ] 環境変数管理（`.env` + `app.config.ts`）
- [ ] ESLint / Prettier 設定

### Phase 2: 認証
- [ ] Firebase 匿名認証
- [ ] 認証状態管理（Zustand）
- [ ] Google ログイン（任意リンク）
- [ ] Apple ログイン（任意リンク）

### Phase 3: コア機能（CRUD）
- [ ] ゴール作成・編集・一覧・完了マーク
- [ ] 曲目 CRUD
- [ ] 練習箇所 CRUD
- [ ] 練習ログ追加（テキスト + 星カウントアップ）

### Phase 4: ダッシュボード
- [ ] 残り日数カウントダウン（直近3件）
- [ ] 各種カウンター表示（ゴール数・曲数・練習箇所数）
- [ ] 練習回数の星表示

### Phase 5: 星演出
- [ ] 段階別の星表示コンポーネント
- [ ] 100回到達時のLottieアニメーション
- [ ] 10回到達時の控えめな演出

### Phase 6: 収益化
- [ ] AdMob 統合（バナー広告）
- [ ] RevenueCat 統合
- [ ] 買い切り課金（広告非表示）
- [ ] 購入復元機能

### Phase 7: リリース準備
- [ ] アプリアイコン・スプラッシュ画面
- [ ] プライバシーポリシー
- [ ] 利用規約
- [ ] App Store / Google Play 申請準備
- [ ] EAS Build 設定

## v1.1 以降（将来構想）

- グラフ・統計機能（週間・月間練習量）
- 演奏会終了後の振り返り機能
- 練習リマインダー通知
- データエクスポート（CSV / PDF）
- 練習音声メモ
- 共有機能（教師との連携）
